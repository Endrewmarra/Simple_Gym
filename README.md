## Como Utilizar este Repositório.

Aqui compartilharemos e manteremos os arquivos e documentos utilizados para este projeto, no momento para o levantamento de requisitos.
Tudo como documentação, imagens e afins ficaram centralizados neste repositório.
Cada um dos envolvidos tem acesso aos arquivos disponíveis aqui e podem modificá-los a vontade, porem para fazer o Upload dos novos arquivos modificados para a pasta principal do projeto sera necessário a verificação com um outro membro do grupo.

---

### Clonar o Repositório

Clonem o repositório do projeto pelo aplicativo do GitHub Desktop ou via linha de comando em seu terminal.

```bash
 git clone https://github.com/Endrewmarra/Gym_System.git
```

Depois de clonado basta acessar a pasta do projeto pela sua IDE (Recomendado VScode)

---

### Instalando Extensões

Neste Projeto foi adicionado uma pasta .VScode para que todos os colaboradores tenham a mesma configuração do projeto para não haver divergências.

No momento só existem duas extensões dentro desta pasta, a code-spell-checker, que serve para sublinhar palavras ortograficamente incorretas, e a code-spell-checker-portuguese-brazilian que adiciona o idioma português a esta extensão.

Em seu Vscode, na busca de extensões (ctrl + shift + x) digite @recommended que as mesmas extensões ja vão aparecer para fazer a instalação.
